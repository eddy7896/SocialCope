# CRITICAL SYSTEM PRINCIPLES

The following principles are NON-NEGOTIABLE.

---

## 1. Semantic-First Architecture

The platform MUST treat all structures as semantic systems.

DO NOT:

* rely on flattened visual-only systems
* treat layouts as raw pixel objects
* use uncontrolled visual blobs

ALWAYS:

* preserve semantic hierarchy
* preserve metadata relationships
* preserve layout inheritance
* preserve semantic scene graph structures

---

## 2. Mutation-Driven Operations

ALL changes MUST pass through the mutation engine.

DO NOT:

* mutate frontend state directly
* manipulate DOM directly
* bypass mutation validation

ALL actions MUST:

* generate structured mutations
* pass validation pipelines
* support rollback systems
* support audit logging

---

## 3. AI Orchestration Over AI Generation

AI acts ONLY as:

* orchestrator
* planner
* semantic assistant
* automation layer

AI MUST NOT:

* generate uncontrolled creative output
* autonomously redesign systems
* bypass tool registries
* manipulate frontend state directly

AI MUST:

* operate deterministically
* use approved tools only
* generate structured operations
* preserve semantic integrity

---

## 4. Local AI Runtime Requirement

The system MUST support:

* local inference
* self-hosted model execution
* modular AI runtimes

Preferred runtimes:

* Ollama
* vLLM
* llama.cpp

Preferred models:

* Qwen 2.5 Coder
* DeepSeek Coder
* Phi-4

DO NOT:

* introduce mandatory cloud AI dependency
* tightly couple workflows to external APIs

---

## 5. Strict Modular Microservices

The system MUST:

* use isolated repositories
* use isolated deployments
* use isolated scaling
* use service boundaries strictly

DO NOT:

* introduce monolithic systems
* collapse service boundaries
* convert architecture into monorepo assumptions

---

## 6. Event-Driven Infrastructure

The system MUST use:

* async workflows
* queue-driven processing
* event-driven orchestration

Preferred technologies:

* Redis Pub/Sub
* BullMQ
* event contracts

---

## 7. Deterministic Systems

ALL automation MUST be deterministic.

DO NOT:

* introduce probabilistic layout behavior
* allow uncontrolled semantic drift
* allow non-repeatable AI workflows
