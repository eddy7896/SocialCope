# Template System Context

Design templates, automation recipes, reusable components.

---

## Templates

Reusable scene graph templates for common layouts.

**Examples:**
- SaaS Dashboard (KPI cards, charts, metrics)
- Campaign Layout (hero, content, CTA)
- Feature Section (image + text)
- Testimonial Section (quotes, avatars)

**Features:**
- Parameterized (customize text, colors)
- Responsive (mobile variants included)
- Design-system aligned
- One-click apply to canvas

---

## Automation Recipes

Workflows for common tasks.

**Examples:**
- "Generate mobile layout" → Responsive generation
- "Organize canvas" → Auto-arrange nodes by type
- "Create KPI dashboard" → Template + styling
- "Apply brand colors" → Design system tokens

---

## Component Library

Reusable semantic components.

**Examples:**
- Button (primary, secondary, disabled states)
- Card (with title, description, footer)
- Form group (label, input, error message)
- Metric card (label, value, trend)

---

## Storage

Templates stored in R2 as JSON + CSS.

---

## Key Files

- `template-spec.md` → Template format
- `recipes.md` → Automation recipes
