# Product Context

Product vision, strategy, personas, success metrics.

---

## Vision

An internal semantic visual OS for rapid design ideation, automation, and collaboration.

**NOT:** Figma competitor. Focused on internal use by easyio teams.

**Goal:** Reduce idea-to-visualization time from hours to minutes.

---

## Users

- **Product teams** → Rapid UI prototyping
- **Marketing teams** → Campaign structure design
- **Systems teams** → Workflow mapping, diagrams
- **Design teams** → Component libraries, automation

---

## Key Metrics

- Time to create layout (target: <5 minutes)
- AI-assisted mutations per project (growth)
- Responsive variant generation success rate
- Export completion time (target: <10 seconds)
- User satisfaction with automation
